# Part-Time Tech

A worker-first directory for verified part-time and fractional technology roles.

The MVP focuses on the job information conventional boards often hide: exact
weekly hours, engagement type, compensation, duration, location, and timezone
overlap. The current role inventory is clearly labeled prototype data.

## Local development

```bash
pnpm install
pnpm run dev
```

## Production build

```bash
pnpm run build
```
